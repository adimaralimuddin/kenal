# KENAL
## Kenal is a social media web app with full features that allows users to stay connected in realtime.

# FEATURES
1. Responsive
2. Dark Theme
3. Realtime
4. Posts
5. Stories
6. Comments
7. Replies
8. Chats
9. Reactions
10. Relations
11. Notification
12. Privacy
13. Settings
14. Blockings
15. Media - Image & Videos

## TECH STACK
1. Javascript
2. ReactJs / NextJS
3. TailwindCss
4. Zustand
5. Firebase firestore
6. Firebase storage
7. Firebase authentication
